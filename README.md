## Overview
The repository contains code used for analysis, plotting, experimental control, and Autodesk Fusion workflows related to the thesis.
The code is provided primarily for transparency, inspection, and reproducibility by readers of the thesis.

## Contents

- `analysis/` — Python code for data analysis and plotting
- `arduino/` — Arduino code used in the experimental setup
- `fusion/` — Python scripts for Autodesk Fusion

## Use of AI assistance

The code in this repository was written with assistance from **GPT-5.5**.

GPT-5.5 was used as a programming assistant for drafting, debugging, restructuring, and documenting code. This includes Python analysis and plotting code, Arduino code, and Autodesk Fusion Python scripts.

All code was reviewed, edited, and tested by the author.

## Notes

Some code may depend on local file paths, external data, specific hardware, Autodesk Fusion configuration, Arduino board configuration, or installed libraries.

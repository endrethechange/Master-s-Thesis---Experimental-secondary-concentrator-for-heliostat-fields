# Full Field Analysis

This folder contains the Python code, raw input images, target file, and selected output files used for full-field target image analysis in the thesis.

## Main scripts

* `analyze_target_series.py` — processes Canon CR2 image series. It detects and warps the target, applies calibration, subtracts matching OFF images where available, and writes heatmaps, debug images, metadata, and summary files.
* `plot_power_vs_irradiance.py` — creates simulated power-versus-irradiance plots from simulation receiver CSV files.

## Supporting modules

* `analysis_detection.py` — target detection, marker refinement, and perspective warping.
* `analysis_geometry.py` — target geometry, marker masks, calibration masks, and analysis masks.
* `analysis_io.py` — input discovery, EXIF parsing, CR2 loading, raw Bayer luminance extraction, and exposure normalization.
* `analysis_measurement.py` — calibration, spot analysis, OFF-image correction, marker filling, and per-image reporting.
* `analysis_comparison.py` — simulation loading, measured/simulated comparison plots, aperture calculations, and power curves.
* `analysis_model.py` — shared constants, filename patterns, physical target dimensions, and data containers.
* `analysis_utils.py` — small helper functions for folders, filenames, display normalization, and coordinate conversion.

## Input files

* Canon CR2 raw images — raw photos used for the full-field analysis.
* `Target/` — contains the SVG file for the five-dot target used in the analysis.

The raw image set contains REF and SEC cases for winter (`01`) and summer (`06`), including ON images, matching OFF images, and NRM calibration/reference images where used.

## Output files

* `analysis_output/` — contains selected output files generated by the analysis pipeline, such as plots, debug images, metadata, and summary results.
* `make_sankey.py` — script placed with the output files to generate Sankey-style SVG diagrams from CSV power values.

## Notes

This code is project-specific and assumes the target geometry, file naming convention, Canon CR2 image format, calibration method, simulation data structure, and scale assumptions used in the thesis.

It is included for transparency, inspection, and documentation of the full-field analysis workflow.

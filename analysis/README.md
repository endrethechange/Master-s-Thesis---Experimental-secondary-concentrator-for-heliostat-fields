# Analysis Code

This folder contains the analysis code used in the master's thesis.

## Folders

* `full_field_analysis/` — code and input/output files for full-field target analysis using Canon CR2 raw images. This includes target detection, image warping, calibration, OFF-frame subtraction, heatmap generation, measured/simulated comparison, and power plots.
* `image_analysis/` — code for camera/image validation tests, including linearity, vignetting, Lambertian response, ISO behavior, and test-rig image analysis.

## Notes

The code is project-specific and assumes the file names, folder structure, camera format, target setup, and calibration methods used in the thesis work.

It is included for transparency, inspection, and documentation of the analysis workflow.

# Autodesk Fusion Scripts

This folder contains Python scripts used in Autodesk Fusion during the master's thesis work.

## Contents

- `HeliostatGeneratorFusion` — places heliostat models in Fusion using position and orientation data from a JSON file.
- `SwapMirrorBodies` — swaps visibility between `Body1` and `Body3` in selected mirror components.
- `MultiSketchToMultiBodyExtrusion` — projects bodies onto a lower plane and extrudes them back up to create base/support geometry.

## Notes

These scripts are project-specific and depend on the Fusion model setup, file names, body names, JSON format, and scale assumptions used in the thesis work.

They are included for transparency and inspection, not as general-purpose Fusion tools.
